Privacy Policy

Effective Date: 10/20/2025

Overview

This Chrome extension respects your privacy. It does not collect, transmit, or share any personal data with third parties. All data used by the extension remains stored locally on your browser.

Data Collected and Stored

The extension stores the following data locally using Chrome's storage API:

  • User preferences and configuration settings
  • A personal GitHub API key entered by the user

This data is used solely to provide and personalize the extension's functionality.

Data Sharing

No data is shared, transmitted, or uploaded to any external server or third party.

The extension does not use analytics, advertising, or tracking technologies.

User Control

You can view or delete your locally stored data at any time by:

  • Removing the extension
  • Clearing extension data via Chrome’s “Clear browsing data” settings.
